Iso 9660 Library 32-Bit

Released 22/01/14

#GBISO9660 allows you to manipulate ISO files. Main functions are 

	- Know if a file exists
	- Extract specific file
	- Extract all the ISO files and folders
	- Get specific file info (size, offset)
